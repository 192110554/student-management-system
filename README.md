# Student Management System

A Full Stack Student Management System built using Spring Boot, MySQL, JavaScript, and Bootstrap.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Search Student by Name
- Pagination (Previous/Next)
- Bootstrap UI
- Toast Notifications

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- Git & GitHub
