# 🎓 Student Management System

A Full Stack Student Management System developed using **Spring Boot**, **MySQL**, **JavaScript**, **Bootstrap**, and **Spring Security**.

This project helps administrators manage students, attendance, dashboard analytics, email notifications, and includes an AI-powered student assistant.

---

- View Student
- Update Student
- Delete Student
- Upload Student Photo
